# Documentation for RxIcon

## Overview
The `RxIcon` component is designed to display icons in Angular applications. It supports various features such as customizable sizes and colors.

## Features
- **Customizable Size**: Easily define the size of the icon.
- **Color Options**: Set the color of the icon to match your application's theme.
- **Responsive Design**: The icon adapts to different screen sizes.

## Usage
To use the `RxIcon`, import it into your Angular module:

```typescript
import { RxIcon } from './path/to/rx-icon';
```

## Props
| Property         | Type             | Description                               |
|------------------|------------------|-------------------------------------------|
| `size`           | `string`         | Defines the size of the icon.            |
| `color`          | `string`         | Defines the color of the icon.           |

## Conclusion
The `RxIcon` component provides a flexible way to display icons in your Angular application. Customize it according to your needs.
