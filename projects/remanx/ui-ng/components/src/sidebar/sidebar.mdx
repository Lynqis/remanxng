# Documentation for RxSidebar

## Overview
The `RxSidebar` component is designed to create a sidebar navigation in Angular applications. It allows for easy organization of links and actions.

## Features
- **Collapsible Sidebar**: Easily toggle the visibility of the sidebar.
- **Customizable Links**: Define links to navigate to different sections of your application.
- **Responsive Design**: The sidebar adapts to different screen sizes.

## Usage
To use the `RxSidebar`, import it into your Angular module:

```typescript
import { RxSidebar } from './path/to/rx-sidebar';
```

## Props
| Property         | Type             | Description                               |
|------------------|------------------|-------------------------------------------|
| `links`          | `Array`          | Defines the links to display in the sidebar. |
| `collapsible`     | `boolean`        | Indicates if the sidebar can be collapsed. |

## Conclusion
The `RxSidebar` component provides a flexible way to create sidebar navigation in your Angular application. Customize it according to your needs.
