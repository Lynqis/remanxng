# Documentation for RxFooter

## Overview
The `RxFooter` component is designed to provide a footer section in Angular applications. It supports various features such as customizable content and actions.

## Features
- **Customizable Content**: Easily define the content displayed in the footer.
- **Action Buttons**: Add buttons for actions directly in the footer.
- **Responsive Design**: The footer adapts to different screen sizes.

## Usage
To use the `RxFooter`, import it into your Angular module:

```typescript
import { RxFooter } from '@lynqis/remanxng/components';
```

## Props
| Property         | Type             | Description                               |
|------------------|------------------|-------------------------------------------|
| `content`        | `string`         | Defines the content of the footer.       |
| `actions`        | `Array`          | Defines action buttons in the footer.    |

## Conclusion
The `RxFooter` component provides a flexible way to create footer sections in your Angular application. Customize it according to your needs.
