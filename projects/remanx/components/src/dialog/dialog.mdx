# Documentation for RxDialog

## Overview
The `RxDialog` component is a customizable dialog box for Angular applications. It supports various features such as confirmation dialogs, custom templates, and responsive design.

## Features
- **Customizable Size**: Set width and height according to your needs.
- **Confirmation Dialogs**: Easily create confirmation dialogs with customizable buttons.
- **Responsive Design**: The dialog adapts to different screen sizes.

## Usage
To use the `RxDialog`, import it into your Angular module:

```typescript
import { RxDialog } from '@lynqis/remanxng/components';
```

## Props
| Property         | Type             | Description                               |
|------------------|------------------|-------------------------------------------|
| `maskVisible`      | `boolean`              | Determines if a background overlay (mask) is visible                        | `true`                                     |
| `transitionOptions`| `string`               | CSS transition for dialog appearance/disappearance                          | `'150ms cubic-bezier(0, 0, 0.2, 1)'`       |
| `closable`         | `boolean`              | Shows a close icon/button in the dialog                                     | `true`                                     |
| `closeOnEscape`    | `boolean`              | Closes the dialog when pressing the Escape key                              | `true`                                     |
| `closeOnOutside`   | `boolean`              | Closes the dialog when clicking outside of it                               | `true`                                     |
| `position`         | `Position`             | Position of the dialog (`'center'`, `'top'`, `'bottom'`, etc.)              | `'center'`                                 |
| `label`            | `string`               | Title or label displayed at the top of the dialog                           | `'Dialog'`                                 |
| `width`            | `string`               | Width of the dialog container (e.g., `'500px'`, `'auto'`)                   | `'auto'`                                   |
| `height`           | `string`               | Height of the dialog container                                              | `'auto'`                                   |
| `confirmDialog`    | `boolean`              | Enables confirmation mode (e.g., for Yes/No dialogs)                        | `false`                                    |


## Events
| Event          | Type                       | Description                                      |
|----------------|----------------------------|--------------------------------------------------|
| `visibleChange`| `EventEmitter<boolean>`     | Emitted when dialog visibility changes           |
| `onClose`      | `EventEmitter<boolean>`     | Emitted when the dialog is closed                |
| `onConfirm`    | `EventEmitter<boolean>`     | Emitted when a confirmation action is validated  |

## Styling
The dialog can be styled using CSS. Refer to the `dialog.css` file for default styles.
