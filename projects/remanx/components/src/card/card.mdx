# Documentation for RxCard

## Overview
The `RxCard` component is designed to display content in a card format in Angular applications. It supports various features such as headers, footers, and customizable styles.

## Features
- **Header and Footer**: Easily add headers and footers to your cards.
- **Customizable Styles**: Apply custom styles to suit your application's theme.
- **Responsive Design**: The card adapts to different screen sizes.

## Usage
To use the `RxCard`, import it into your Angular module:

```typescript
import { RxCard } from './path/to/rx-card';
```

Refer to the following stories for examples of how to implement the `RxCard`:

## Props
| Property         | Type             | Description                               |
|------------------|------------------|-------------------------------------------|
| `header`         | `string`         | Defines the header text of the card.     |
| `footer`         | `string`         | Defines the footer text of the card.     |

## Events
- **onCardClick**: Emitted when the card is clicked.

## Styling
The card can be styled using CSS. Refer to the `card.css` file for default styles.

## Conclusion
The `RxCard` component provides a flexible way to display content in card format in your Angular application. Customize it according to your needs and refer to the provided stories for implementation examples.
