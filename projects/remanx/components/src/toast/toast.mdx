# Documentation for RxToast

## Overview
The `RxToast` component is designed to display brief messages or notifications in Angular applications. It supports various features such as customizable durations, positions, and types.

## Features
- **Customizable Duration**: Set how long the toast message should be displayed.
- **Position Options**: Choose where the toast will appear on the screen.
- **Different Types**: Display different types of messages (success, error, info).

## Usage
To use the `RxToast`, import it into your Angular module:

```typescript
import { RxToast } from '@lynqis/remanxng/components';
```

## Props
| Property     | Type                         | Description                                                             | Default     |
|--------------|------------------------------|-------------------------------------------------------------------------|-------------|
| `position`   | `Position`                   | Position of the toast (`'top'`, `'bottom'`, etc.)                       | `'top'`     |
| `severity`   | `Severity`                   | Visual severity style (`'primary'`, `'success'`, `'warn'`, etc.)        | `'primary'` |
| `lifetime`   | `number`                     | Duration (in seconds) each toast message remains visible                | `120`       |
| `messages`   | `ToastMessage[] \| null \| undefined` | List of toast messages to display                                | `undefined` |

## Events
- **onClose**: Emitted when the toast is closed.

## Styling
The toast can be styled using CSS. Refer to the `toast.css` file for default styles.
