# Documentation for RxButton

## Overview
The `RxButton` component is designed to create customizable buttons in Angular applications. It supports various styles, sizes, and click events.

## Features
- **Customizable Styles**: Define styles for different button states (hover, active, disabled).
- **Size Options**: Choose from small, medium, and large button sizes.
- **Loading State**: Display a loading spinner while processing actions.

## Usage
To use the `RxButton`, import it into your Angular module:

```typescript
import { RxButton } from '@lynqis/remanxng/components';
```

## Props
| Property     | Type                | Description                                                                 | Default       |
|--------------|---------------------|-----------------------------------------------------------------------------|---------------|
| `autofocus`  | `boolean`           | Automatically focuses the button on load                                    | `false`       |
| `disabled`   | `boolean`           | Disables the button                                                         | `false`       |
| `label`      | `string`            | Text label displayed inside the button                                      | `undefined`   |
| `loading`    | `boolean`           | Shows a loading state (e.g., spinner)                                       | `false`       |
| `name`       | `string`            | Button name attribute                                                       | `undefined`   |
| `severity`   | `Severity`          | Defines the visual importance or style of the button (e.g., primary, info)  | `undefined`   |
| `type`       | `string`            | HTML button type: `"button"`, `"submit"`, `"reset"`                         | `'button'`    |
| `value`      | `string`            | Value attribute of the button                                               | `undefined`   |
| `hidden`     | `boolean`           | Hides the button visually (uses `hidden` attribute)                         | `false`       |
| `noStyle`    | `boolean`           | Disables default styling                                                    | `false`       |


## Events
- **onClick**: Emitted when the button is clicked.

## Styling
The button can be styled using CSS. Refer to the `button.css` file for default styles.
